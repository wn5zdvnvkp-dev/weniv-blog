📘 My Personal Blog Project
---
사용자 경험을 고려한 깔끔하고 현대적인 디자인의 개인 블로그 웹사이트입니다. 시안을 바탕으로 한 정교한 UI 구현과 반응형 레이아웃을 목표로 제작되었습니다.

---

🚀 프로젝트 개요
진행 상태: UI/UX 퍼블리싱 완료 및 API 연동 준비 단계

주요 특징:

배너와 본문이 겹치는 트렌디한 레이아웃 적용

사용자 상태(로그인/비로그인)에 따른 가변형 헤더 시스템

모달(Modal)을 활용한 삭제 확인 인터랙션 구현

---

📂 파일 구조
```
.
├── index.html          # 메인 페이지 (포스트 카드 리스트)
├── login.html          # 로그인 페이지
├── signup.html         # 회원가입 페이지
├── mypage.html         # 마이페이지 (프로필 및 회원정보)
├── post-detail.html    # 포스트 상세 조회 페이지
├── post-write.html     # 포스트 작성/수정 페이지
├── css/
│   ├── reset.css       # 브라우저 기본 스타일 초기화
│   ├── style.css       # 공통 스타일 및 메인 레이아웃
│   ├── post.css        # 포스트 관련 (상세, 작성) 전용 스타일
│   └── auth.css        # 인증 관련 (로그인, 회원가입) 스타일
├── js/
│   ├── api.js          # API 통신 모듈
│   ├── auth.js         # 인증 로직 (로그인/로그아웃)
│   ├── post.js         # 포스트 CRUD 관련 로직
│   ├── main.js         # 메인 페이지 인터랙션
│   └── utils.js        # 공통 유틸리티 함수
└── assets/             # 아이콘, 배너, 프로필 등 이미지 자산
```

---

✨ 주요 기능 및 특징
1. UI/UX 디자인

Banner System: 페이지별 특화된 배너 시스템 (날짜 자동 위젯 및 배경 이미지 커스텀).

Interactive UI: 상세 페이지 내 '뒤로가기' 버튼의 부유형 배치 및 포스트 카드 호버 효과.

Write Form: textarea 리사이즈 제한 및 최소 높이(min-height: 720px) 설정을 통한 안정적인 글쓰기 환경 제공.

2. 컴포넌트

Navigation: 로그인 상태에 따라 프로필 이미지, 글쓰기 버튼, 로그아웃 버튼이 노출되는 동적 내비게이션 바.

Alert Modal: 삭제 전 사용자 확인을 위한 커스텀 모달 팝업 구현.

Top Button: 페이지 최하단에서 최상단으로 부드럽게 이동하는 스크롤 기능.

3. 기술 스택

HTML5 / CSS3: 시맨틱 마크업 및 Flex/Grid 레이아웃 활용.

JavaScript (Vanilla JS): 외부 라이브러리 없이 순수 자바스크립트를 이용한 기능 구현 및 API 연동 설계.
